Additional scripts can be found in /views/assets/js/:
- bareTransport.js: Locally installed module for Ultraviolet functionality.
- card.js: Adds visual effects to the box cards.
- common.js: Provides essential features across all pages.
- csel.js: Manages the settings menu.
- particles.js: Creates animated background effects.
- register-sw.js: Handles service workers and Ultraviolet integration.
